Synthèse des directions
D1 + D2 — Direction principale : carte full-bleed, corridor slider overlay, bottom sheet Vaul pour les POI. La carte n'est jamais masquée.
D3 + D4 — Mode Live cohérent avec la direction principale : interface épurée, consentement RGPD explicite.
D5 — États dégradés systématisés : toujours informatif, jamais alarmiste, données maintenues visibles.
D6 — Variante navigation : bottom tab bar vs. FAB flottant. Question ouverte : D6 est plus familier iOS, mais consomme 56px de la carte.

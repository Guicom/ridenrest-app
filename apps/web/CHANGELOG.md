# Changelog

## 1.0.0 — 2026-04-09

### Nouveautés
- Système de notes de version avec popin au lancement
- Aventures multi-segments avec import GPX et Strava
- Carte interactive avec recherche POI par corridor
- Analyse de densité d'hébergements colorisée
- Mode live avec géolocalisation et recherche en temps réel
- Planification par étapes avec météo ajustée
- PWA installable avec support offline partiel

### Améliorations
- Liens Booking.com enrichis avec ville, code postal, région et pays
- Pins colorés par catégorie de POI sur la carte
- Profil d'élévation interactif avec synchronisation carte
- Sidebar mobile responsive avec bouton flottant

### Corrections
- Correction du mode live qui ignorait Google Places comme source primaire
- Correction de l'auto-zoom en mode planning avec cache chaud
